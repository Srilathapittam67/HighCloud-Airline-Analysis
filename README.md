# ✈️ High Cloud Airline Analysis Dashboard

## 📌 Project Overview
This project analyzes airline data to understand performance, passenger trends, and operational efficiency using interactive dashboards.
## 📊 Dataset
Due to GitHub size limitations, the dataset is hosted externally.
🔗 **[Download Dataset](https://docs.google.com/spreadsheets/d/10neIdPrzsBOBFBWReBiUlFthY_Dz3XZw/edit?usp=sharing)**
## 🛠 Tools & Technologies
- Power BI  
- Excel  
- Data Cleaning  
- Data Visualization  
## 📈 Key KPIs
- Load Factor Analysis (Yearly, Monthly trends)  
- Passenger Count by Airline  
- Top 10 Airlines Performance  
- Route & Distance Analysis  
## 💡 Key Insights
- Identified top airlines based on passenger volume  
- High load factor indicates better seat utilization  
- Found trends in airline performance across regions
